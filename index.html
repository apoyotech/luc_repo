Instalación Installation

https://apoyotech.github.io/luc_repo/repo/
  1. 1

    En Kodi: Ajustes → Sistema → Addons y activa Orígenes desconocidos.

    In Kodi: Settings → System → Add-ons and enable Unknown sources.

  2. 2

    Ajustes → Administrador de archivos → Añadir fuente, pega la URL de arriba y dale un nombre (por ejemplo, luc_repo).

    Settings → File manager → Add source, paste the URL above and give it a name (for example, luc_repo).

  3. 3

    Vuelve a Addons y elige Instalar desde archivo zip, abre la fuente que acabas de añadir y selecciona el ZIP del repositorio.

    Go back to Add-ons and choose Install from zip file, open the source you just added and select the repository ZIP.

  4. 4

    Instala los addons desde Instalar desde repositorio → luc_repo.

    Install the add-ons from Install from repository → luc_repo.
